FB Lead Scraper - Settings

Facebook Groups

Select which groups to crawl. Only checked groups will be included in each crawl.

Posts Per Group

How many additional pages to fetch beyond the initial page load (each page (appx) 3 posts). Keep low to avoid detection.

Crawl Schedule

Set to 0 for manual-only crawling (use the "Crawl Now" button in the popup).

Rate Limiting

Random pause between requests to mimic human browsing. A value is picked randomly between min and max each time.

Advanced

Level 1 Keywords (OR matching)

Posts matching ANY of these keywords pass level 1. One keyword per line. Multi-word phrases are supported.

Exclude Keywords

Posts containing ANY of these are skipped immediately (marked as seen so they don't reappear). One per line.

Level 2+ Keyword Groups (AND matching)

Applied after level 1. A post must contain ALL keywords in at least one enabled group to pass. The "All" group is a pass-through - when enabled, everything passes. Groups are OR'd between each other, AND'd within.
